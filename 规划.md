# S-TextPaste 规划文档

## 项目定义

绝密级零信任端到端加密文本分享服务。所有加解密在浏览器完成，服务端仅存储密文。

- 仓库：`https://github.com/stop666two/S-TextPaste`
- 部署：Cloudflare Workers + D1，一键按钮部署

## 加密方案

### 密码派生密钥 (512-bit, 12轮)
1. SHA-256(pw||salt) → MD5(pw||salt) → 链式迭代12轮
2. 每轮 SHA ⊕ MD5 XOR 交织
3. 64字节输出，前32字节作AES-256密钥

### 后量子密钥A (1024-bit, 24轮)
- SHA-256(子盐)主导，MD5嵌套，链式迭代

### 后量子密钥B (1024-bit, 24轮)
- MD5主导，SHA嵌套，与A完全逆向推导

### 三重包络 + HMAC
1. DEK(随机) → AES-256-GCM 加密明文
2. KeyA(1024bit) → AES-256-GCM 加密DEK
3. KeyB(1024bit) → AES-256-GCM 加密上一步
4. HMAC-SHA-256(integrityKey, 全载荷)

## 数据库 (Cloudflare D1)

```sql
CREATE TABLE pastes (
  id TEXT PRIMARY KEY, mode TEXT NOT NULL, salt TEXT,
  encrypted_payload TEXT NOT NULL, hint TEXT DEFAULT '',
  delete_token_hash TEXT NOT NULL, expires_at INTEGER,
  max_views INTEGER DEFAULT -1, view_count INTEGER DEFAULT 0,
  burn_after_read INTEGER DEFAULT 0, created_at INTEGER NOT NULL,
  pubkey_fingerprint TEXT
);
```

## API

| 方法 | 路径 | 请求 | 返回 |
|------|------|------|------|
| POST | /api/paste | mode, encrypted_payload, ... | id, delete_token |
| GET | /api/paste/:id | - | encrypted_payload, view_count... |
| POST | /api/paste/:id/view | - | success |
| DELETE | /api/paste/:id | X-Delete-Token | success |

## 前端页面

- **CreatePage**: CodeMirror 6 编辑器 + 实时预览 + 加密配置面板 + 生命周期设置
- **ReadPage**: 解密表单 + 安全性仪表盘 + 密码提示 + 5次锁定
- **ViewPage**: Markdown 渲染(含 Mermaid/KaTeX/代码高亮) + 安全仪表盘 + 删除面板

## 安全

- AES-256-GCM · HMAC-SHA-256 · CSP
- 32字符随机ID (64^32) · 速率限制 · 删除令牌SHA-256哈希
- content不在URL · React Router state内存传递
